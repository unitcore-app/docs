Cluster at capacity

503

cluster.at_capacity

The cluster does not have sufficient capacity to provision this resource.

Common causes

  • No worker node has enough CPU / memory / storage to schedule the new pod.
  • Storage class has hit its provisioned quota.

How to resolve

  • Scale the cluster (add a worker, increase Longhorn capacity).
  • Retry once cluster operators have grown capacity.
