Errors  /  Scraping

Invalid tenant id

400

scraping.invalid_tenant_id

Tenant id is not a valid identifier for any known tenant.

Common causes

  • Wrong tenant id format.
  • Tenant was deleted or renamed.

How to resolve

  • Confirm the tenant id from Hub.
  • Stop sending traffic for deleted tenants.
