Errors  /  Scraping

Body too large

413

scraping.body_too_large

The scraping payload exceeds the maximum allowed size.

Common causes

  • Single batch is too large.
  • Embedded blobs not compressed.

How to resolve

  • Split the payload into smaller batches.
  • Compress large fields before sending.
