Invalid state transition

409

resource.invalid_state_transition

The requested state change is not allowed from the current state.

Common causes

  • The workflow definition does not allow this transition (e.g. `draft → archived` without going through `published`).
  • The resource is locked by another process.

How to resolve

  • Fetch the resource and inspect its current status.
  • Follow the workflow path defined by the owning module.
