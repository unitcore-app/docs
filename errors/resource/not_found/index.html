Resource not found

404

resource.not_found

The requested resource was not found.

Common causes

  • The id in the URL does not exist for the current tenant.
  • The record was soft-deleted (`is_deleted = true`).
  • The caller is in the wrong site context and the record belongs to a different site.

How to resolve

  • Verify the id is correct and current.
  • Confirm the request includes the right `X-Site-Id` (or equivalent) header.
  • If you expect the record to still exist, check the audit log for a delete event.
