Constraint violation

409

resource.constraint_violation

The request would violate a data integrity constraint.

Common causes

  • A foreign key would dangle.
  • A check constraint was violated.
  • A delete was attempted against a record still referenced by others.

How to resolve

  • Remove or update referencing records first.
  • Inspect the `detail` field of the response for the exact constraint name.
