Duplicate resource

409

resource.duplicate

A resource with the same unique value already exists.

Common causes

  • A unique constraint was hit (slug, email, code).
  • A concurrent request created the same record a moment earlier.

How to resolve

  • Read the existing record before creating a new one.
  • Use idempotency keys for safe retries.
  • Pick a different unique value (slug, code) and retry.
