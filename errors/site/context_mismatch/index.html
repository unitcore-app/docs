Site context mismatch

403

site.context_mismatch

The request's site context does not match the resource's site.

Common causes

  • Attempted to read or modify a record from a different tenant.
  • Token's site claim does not match the `X-Site-Id` header.
  • User assumed a site they are not a member of.

How to resolve

  • Confirm the site id on both the token and the request header match the resource's site.
  • Re-authenticate with the correct site selected.
