Site required

400

validation.site_required

The endpoint requires a site context and none was provided.

Common causes

  • Missing site selector on a multi-tenant endpoint.
  • Token issued without a site claim.

How to resolve

  • Include the site identifier on the request.
  • Re-issue the token with the desired site selected.
