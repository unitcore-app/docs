Invalid dev bypass

400

validation.invalid_dev_bypass

The development-only authentication bypass header is malformed or used outside development mode.

Common causes

  • Header `X-Dev-Bypass` was sent in a non-development environment.
  • The bypass value does not match the configured development secret.

How to resolve

  • Remove the dev-bypass header from production requests.
  • In development, ensure the header value matches `DevAuth:BypassToken` from configuration.
