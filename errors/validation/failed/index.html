Validation failed

400

validation.failed

One or more validation errors occurred.

Common causes

  • A required field was omitted from the request body.
  • A field value did not match its type (e.g. string where GUID expected).
  • A composite rule across multiple fields was violated.

How to resolve

  • Read the `errors` array on the problem+json response; each entry names the failing field and rule.
  • Re-submit the request with corrected field values.
  • If the validation rule looks wrong, file an issue against the owning module.
