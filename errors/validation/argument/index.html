Errors  /  Validation

Invalid argument

400

validation.argument

One or more arguments are invalid.

Common causes

  • Query-string parameter failed model binding.
  • A route value did not match its constraint.

How to resolve

  • Verify all query-string and route parameters match the API contract.
  • URL-encode values that contain reserved characters.
