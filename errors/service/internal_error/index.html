Internal server error

500

service.internal_error

An unexpected error occurred. Please try again later.

Common causes

  • An unhandled exception was raised by a module.
  • A bug surfaced for an unanticipated input.

How to resolve

  • Retry once after a short delay.
  • Include the `traceId` from the response when filing a support ticket.
