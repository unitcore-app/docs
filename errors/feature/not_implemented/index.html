Not implemented

501

feature.not_implemented

The endpoint exists but its underlying feature is not implemented in the current build.

Common causes

  • Feature module is disabled in this deployment mode.
  • Endpoint is a placeholder for a planned feature.

How to resolve

  • Enable the owning module in `Modules__Enabled` (if available).
  • Track the feature roadmap for completion.
