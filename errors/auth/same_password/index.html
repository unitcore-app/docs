Errors  /  Authentication & authorization

Same password

400

auth.same_password

The new password must differ from the current password.

Common causes

  • The change-password request supplied a new password identical to the existing one.

How to resolve

  • Choose a new password that is different from the current one and any disallowed-by-history values.
