Errors  /  Authentication & authorization

Invalid password

400

auth.invalid_password

The current password is incorrect.

Common causes

  • User entered the wrong existing password during a change-password flow.

How to resolve

  • Verify the current password.
  • Use the password-reset flow if it is forgotten.
