Errors  /  Authentication & authorization

Authentication required

401

auth.unauthorized

Authentication is required to access this resource.

Common causes

  • Missing `Authorization` header.
  • Session cookie expired or was cleared.
  • Calling a protected endpoint anonymously.

How to resolve

  • Sign in and retry with a valid token / cookie.
  • If using a service principal, confirm its credentials are still active.
