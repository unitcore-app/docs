Agent key missing site

400

auth.agent_key_no_site

The agent key is valid but is not bound to a site, and the endpoint requires one.

Common causes

  • Agent key was created without a site binding.
  • Calling a tenant endpoint with a global agent key.

How to resolve

  • Re-issue the key with the target site bound to it.
  • Use a site-scoped key for tenant endpoints.
