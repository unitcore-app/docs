Session invalid

401

auth.session_invalid

The current session is no longer valid.

Common causes

  • Session was terminated server-side.
  • Session belongs to a different device fingerprint than the request.

How to resolve

  • Sign in again.
  • If sessions are unexpectedly being invalidated, check the session policy on the site.
