MFA required

401

auth.mfa_required

Two-factor authentication is required to complete sign-in.

Common causes

  • The account has MFA enabled and the second factor has not been provided.
  • The site enforces MFA for all users.

How to resolve

  • Submit the MFA challenge response (`/api/v1/auth/mfa/verify`).
  • If the user lost their authenticator, an admin can reset the MFA enrollment.
