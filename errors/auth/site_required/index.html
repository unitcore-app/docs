Site context required

400

auth.site_required

The request must include a valid site context.

Common causes

  • Missing `X-Site-Id` header on a multi-tenant endpoint.
  • Token was issued without a site claim.

How to resolve

  • Include the correct site identifier on the request.
  • Re-issue the token with the desired site selected.
