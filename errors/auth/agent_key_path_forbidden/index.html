Agent key path forbidden

403

auth.agent_key_path_forbidden

The agent key is valid but is not allowed to call this path.

Common causes

  • The key's `allowedPaths` allow-list does not match the request path.
  • Trying to use an agent key on a user-only endpoint.

How to resolve

  • Update the key's `allowedPaths` to include this endpoint.
  • Use a different key (or a user token) with broader access.
