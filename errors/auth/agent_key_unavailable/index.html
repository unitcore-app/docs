Errors  /  Authentication & authorization

Agent key unavailable

503

auth.agent_key_unavailable

The agent-key authentication backend could not be reached.

Common causes

  • The agent key service is down or starting up.
  • Network policy blocks the API from the agent key store.

How to resolve

  • Retry after a short delay.
  • Operator: check the agent key service health.
