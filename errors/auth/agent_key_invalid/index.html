Agent key invalid

401

auth.agent_key_invalid

The presented agent key is not recognized or has been revoked.

Common causes

  • Key was deleted or rotated.
  • Key was minted for a different environment.

How to resolve

  • Mint a fresh agent key from the correct environment.
  • Re-deploy the agent with the new key.
