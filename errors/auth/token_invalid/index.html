Errors  /  Authentication & authorization

Token invalid

401

auth.token_invalid

The authentication token is invalid.

Common causes

  • Token signature did not validate.
  • Token was issued for a different audience or site.
  • Token has been revoked.

How to resolve

  • Sign in again to obtain a fresh token.
  • If you are forwarding a token from another service, verify the audience matches this API.
