Errors  /  Authentication & authorization

Invalid credentials

401

auth.invalid_credentials

The email or password is incorrect.

Common causes

  • Wrong email or password.
  • Account exists on a different site.
  • Account was migrated and password hash format has changed.

How to resolve

  • Confirm the email is correct and registered on this site.
  • Use the password-reset flow if the password is forgotten.
