Token expired

401

auth.token_expired

Your session has expired. Please sign in again.

Common causes

  • Access token lifetime elapsed.
  • Refresh token revoked.

How to resolve

  • Refresh the token via `/api/v1/auth/refresh`.
  • If refresh fails, sign the user in again.
