Portal audience not allowed

403

auth.portal_audience_not_allowed

The token was issued for a different audience than the endpoint accepts.

Common causes

  • Hub portal token used against a tenant API (or vice versa).
  • Audience claim was rewritten by a misconfigured proxy.

How to resolve

  • Obtain a token whose `aud` claim matches the target API.
  • Stop forwarding tokens across audience boundaries.
