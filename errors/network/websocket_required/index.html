WebSocket required

400

network.websocket_required

This endpoint requires a WebSocket upgrade and the client sent a plain HTTP request.

Common causes

  • Calling a SignalR / WebSocket endpoint with `fetch` or `curl`.
  • Proxy stripped the `Upgrade` header.

How to resolve

  • Use a WebSocket client.
  • Configure intermediaries to forward `Upgrade` and `Connection` headers.
