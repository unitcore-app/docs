Errors  /  CORS

Origin not allowed

403

cors.origin_not_allowed

The browser origin is not on the allow-list for this API.

Common causes

  • Frontend hosted on a domain not registered with the site.
  • Misconfigured CORS allow-list on the API.

How to resolve

  • Add the origin to the site's CORS allow-list.
  • Use a same-origin request through your own backend proxy.
