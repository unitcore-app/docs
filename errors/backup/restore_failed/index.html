Errors  /  Backup

Restore failed

500

backup.restore_failed

The restore operation did not complete successfully.

Common causes

  • Disk error during restore.
  • Schema drift between the backup and the current database.

How to resolve

  • Check server logs for the underlying error.
  • If schema drift is suspected, restore to a sidecar database and migrate forward.
