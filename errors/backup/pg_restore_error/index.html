Errors  /  Backup

pg_restore error

500

backup.pg_restore_error

pg_restore returned an error. Check server logs for details.

Common causes

  • Backup file format incompatible with the running pg_restore.
  • Object ownership or extension mismatch.

How to resolve

  • Match pg_restore version to the backup origin.
  • Drop incompatible objects in the target before restoring.
