Errors  /  Backup

Backup already running

409

backup.in_progress

A backup is already running. Please wait for it to finish.

Common causes

  • An earlier backup job is still active for this tenant.
  • A scheduled backup overlapped with a manual one.

How to resolve

  • Wait for the active backup to complete and retry.
  • Check the backups list to see the in-flight job.
