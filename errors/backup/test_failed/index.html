Backup test failed

500

backup.test_failed

Backup restoration test encountered an error.

Common causes

  • The test restore environment could not be spun up.
  • The backup file is corrupt.

How to resolve

  • Inspect the backup test logs.
  • Re-run a fresh backup and test it again.
