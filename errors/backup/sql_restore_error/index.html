SQL restore error

500

backup.sql_restore_error

SQL restore was rolled back due to an error.

Common causes

  • A statement in the SQL dump failed (constraint, missing role, etc.).
  • Target database is not empty.

How to resolve

  • Inspect the restore log for the failing statement.
  • Restore into an empty database and migrate data forward.
