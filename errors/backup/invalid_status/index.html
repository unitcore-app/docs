Invalid backup status

409

backup.invalid_status

Only completed backups can be restored.

Common causes

  • Attempted to restore a backup that is still running or has failed.

How to resolve

  • Wait until the backup reaches `Completed` status, then restore.
  • If the backup failed, run a fresh backup first.
