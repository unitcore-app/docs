Backup file missing

404

backup.file_not_found

The backup file could not be located on disk.

Common causes

  • The backup record points to a file that has been deleted from storage.
  • Storage volume was reset or replaced without re-uploading backups.

How to resolve

  • Check the storage backend for the file.
  • If the file is permanently lost, mark the backup record as failed and create a fresh backup.
