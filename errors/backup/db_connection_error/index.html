Database connection error

500

backup.db_connection_error

Could not connect to the database to perform restore.

Common causes

  • Database is down or unreachable.
  • Restore credentials are wrong.

How to resolve

  • Verify the database is up and the restore credentials are valid.
  • Check network policies between the Hub and the target database.
