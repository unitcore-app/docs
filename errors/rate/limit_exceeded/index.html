Errors  /  Rate limiting

Rate limit exceeded

429

rate.limit_exceeded

You have exceeded the request rate for this endpoint.

Common causes

  • Too many requests in a short window.
  • A single client is hammering an endpoint without backoff.

How to resolve

  • Honour the `Retry-After` response header.
  • Add client-side rate limiting and exponential backoff.
